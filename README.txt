# Planiform V3
> Courte description du logiciel.

Ce logiciel est un outil pédagogique complet conçu pour aider les formateurs à gérer efficacement leurs cours. Il offre une variété de fonctionnalités, dont l'importation des objectifs pédagogiques, la planification des contrôles de module, la génération de préparations pédagogiques, et bien plus encore. Grâce à une interface conviviale, les formateurs peuvent générer rapidement et facilement les documents nécessaires pour leurs cours.
Ce logiciel a été développé par Dr. Ali Sadiqui en collaboration avec l'Ordre National des Docteurs de l'OFPPT.

## Caractéristiques du Logiciel

Le logiciel offre une variété de fonctionnalités conçues pour optimiser et faciliter le processus d'enseignement :

1. **Importation des Modules et de leurs Objectifs Pédagogiques :** Le logiciel est capable d'importer des modules ainsi que leurs objectifs pédagogiques directement à partir de guides pédagogiques, réduisant ainsi le temps passé à saisir les données manuellement.

2. **Décomposition des Objectifs Pédagogiques :** En tenant compte de la masse horaire du module, le logiciel décompose efficacement l'ensemble des objectifs pédagogiques en un nombre approprié de séances.

3. **Planification des Contrôles de Module :** Les formateurs peuvent personnaliser la planification de leurs contrôles de module, offrant une flexibilité maximale pour répondre aux besoins spécifiques de chaque cours.

4. **Génération des Préparations Pédagogiques :** Le logiciel génère automatiquement des préparations pédagogiques pour les séances en présentiel et à distance, garantissant la cohérence et l'efficacité de l'enseignement.

5. **Création du Classeur Pédagogique du Formateur :** Pour chaque groupe, le logiciel est capable de générer un classeur pédagogique personnalisé pour le formateur, ce qui facilite l'organisation et la gestion des tâches pédagogiques.

6. **Utilisation de ChatGPT pour la Préparation Pédagogique :** Avec l'intégration de ChatGPT, les formateurs peuvent obtenir une aide précieuse pour remplir leurs préparations pédagogiques, ce qui rend le processus encore plus efficace et convivial.

## Installation

Le logiciel est compatible avec Windows 10 et les versions ultérieures. Il requiert l'installation de Java 1.8 ou une version plus récente pour fonctionner correctement.
## Comment l'utiliser

Le programme est conçu comme un assistant pour faciliter son utilisation. Il suffit de suivre les instructions à l'écran pour générer les documents souhaités.

## utiliser l'IA pour gérer directement du texte

# Le texte à utiliser dans l'IA pour gérer la motivation :
"À partir des objectifs pédagogiques suivants :" [saisir les objectifs de la séance].
"rédiger, en deux phrases maximum, un texte de motivation pour les stagiaires. Commencez par : 'Dans cette séance, nous allons...'. Rédigez en français."

# Le texte à utiliser dans l'IA pour gérer la synthèse de la séance :
"À partir des objectifs pédagogiques suivants :" [saisir les objectifs de la séance].
"rédiger, en deux phrases maximum, un texte qui permet de faire une synthèse de la séance pour les stagiaires. Commencez par : 'Dans cette séance, nous avons vu...'. Rédigez en français."
